# Bilibili Speed Master | B站倍速大师

## 🚀 简介

这是一个专为 Bilibili 设计的轻量级浏览器扩展（基于 Manifest V3），旨在突破 B 站网页端原生播放器的倍速限制。

**为什么需要它？**

- **突破极限**：B 站官方最高仅支持 2.0x，本插件支持 **0.1x - 16x** 自由调节。
- **高效学习**：大三党刷课、看技术讲座必备，3.0x 或 4.0x 快速过知识点。
- **极简代码**：无后台常驻，不占用内存，零权限滥用。

------

## 🛠️ 技术栈

- **Manifest V3**: 遵循谷歌最新的浏览器插件规范。
- **Native JavaScript**: 纯原生 JS 操作 DOM 与 HTML5 Video 接口。
- **Chrome Scripting API**: 动态注入脚本实现跨域控制。

------

## 📦 安装教程

由于本插件尚未上架 Chrome Web Store，请使用“开发者模式”手动加载：

1. **下载代码**：点击本仓库右上角的 `Code` -> `Download ZIP` 并解压。
2. **打开扩展页面**：
   - **Chrome**: 地址栏输入 `chrome://extensions/`
   - **Edge**: 地址栏输入 `edge://extensions/`
3. **开启开发者模式**：打开页面右上角或左侧的 **“开发者模式”** 开关。
4. **加载插件**：点击 **“加载解压缩的扩展”**，选择你解压后的文件夹（包含 `manifest.json` 的那个）。
5. **固定插件**：在浏览器工具栏点击“拼图”图标，将 **B站自定义倍速** 固定。

------

## 📖 使用说明

1. 打开任意一个 [Bilibili](https://www.bilibili.com) 视频播放页面。
2. 点击插件图标，在输入框输入你想要的倍速（支持小数点，如 `2.5`）。
3. 点击 **“应用倍速”**，视频将立即按新速度播放。
4. 点击 **“恢复 1x”** 即可一键重置。
